<h3 id="flatrecursivearray"><code>flatRecursive(array)</code></h3>

Flattens an array of nested arrays using recursion

#### Arguments
1. `array` *(Array)*: input array

#### Returns
*(Array)*: the flattened array

#### Example
```js
const result = flatRecursive([1, [2, [3, [4]]]]);
console.log(result);
> [1, 2, 3, 4]
```
